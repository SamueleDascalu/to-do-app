# TODO APP

Functionalities:
  - light/dark mode
  - add new task
  - delete tasks
  - mark tasks as completed

Colors:

*light mode:

  - base (gray): #EEEEEE
  - azure: #76ABAE
  - light black: #31363F
  - black: #222831

*dark mode:

  - gray: #EEEEEE
  - azure: #76ABAE
  - light black: #31363F
  - base (black): #222831
